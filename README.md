RoyalGames API
A RoyalGames API é um sistema de gerenciamento de catálogo de jogos desenvolvido em .NET, focado em organização por gêneros, plataformas e classificações indicativas. O sistema conta com um robusto sistema de Auditoria de Alterações e segurança via JWT.

Linguagem: C#

Framework: .NET 8.0

ORM: Entity Framework Core

Banco de Dados: SQL Server

Documentação: Swagger

Autenticação: JWT
